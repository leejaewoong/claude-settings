## 디자인 방향 예시

아래는 SKILL.md의 "Tone" 중 게임 UI에 적합한 방향 예시이다. 매 생성마다 다른 방향을 선택한다:

```yaml
방향_예시:
  네온_사이버펑크: "어두운 배경 + 네온 글로우 + 스캔라인 + 기하학 형태"
  브루탈리스트: "극단적 대비 + 거친 테두리 + 모노스페이스 레이블 + 그리드 노출"
  글래스모피즘: "다층 블러 패널 + 미묘한 반사 + 부드러운 그림자 + 투명 계층"
  레트로_퓨처: "CRT 효과 + 오래된 기술 미학 + 그리드 라인 + 앰버/그린 모니터"
  미니멀_럭셔리: "극도의 여백 + 가는 라인 + 모노크롬 + 단 하나의 액센트"
  아트_데코: "기하학 패턴 + 금속 질감 + 대칭 구성 + 장식 프레임"

  주의: "매번 같은 방향을 반복하지 않는다. 다양성이 실험적 안의 핵심 가치."
```
