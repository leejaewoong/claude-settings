# CoachMark (코치 마크)

## 용도
온보딩 가이드 마크. 신규 기능 하이라이트, 사용 안내 등.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| Direction | Top | 위에서 가리킴 |
| | Bottom | 아래에서 가리킴 |
| | Left | 왼쪽에서 가리킴 |
| | Right | 오른쪽에서 가리킴 |

## 추천 사이즈

| 항목 | 값 |
|------|-----|
| 너비 | 200~320 |
| 높이 | 60~120 |

## textOverrides

| 키 | 설명 |
|----|------|
| Title | 가이드 제목 |
| Description | 가이드 설명 |

## 색상 정보

```yaml
색상:
  배경: "#2a2a35"
  border: "#3a3a45"
  cornerRadius: 8
  화살표: "#2a2a35"
  타이틀: "#eaeaea"
  설명: "#8b8b8b"
  강조: "#f2a900"
```

## 규칙 (위키 사양서)

### 용도
- 신규 콘텐츠/기능 최초 진입 시 안내
- 한 화면에 1개만 표시 권장

### 배치
- 안내 대상 UI 요소 근처에 화살표 방향으로 배치
