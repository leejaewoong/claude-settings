# FilterPopup (필터 팝업)

## 용도
필터 팝업 패널. 다중 필터 선택, 고급 검색 필터 등.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| Type | Default | 기본 필터 |
| | Multi | 다중 선택 필터 |

## 추천 사이즈

| 항목 | 값 |
|------|-----|
| 너비 | 280~400 |
| 높이 | 가변 (200~600) |
| 필터 항목 높이 | 36~44 |

## 색상 정보

```yaml
색상:
  배경: "#1e1e24"
  border: "#2a2a35"
  cornerRadius: 8
  shadow: "0 4px 16px rgba(0,0,0,0.5)"
  타이틀: "#eaeaea"
  필터_라벨: "#8b8b8b"
  필터_값_선택: "#eaeaea"
  필터_값_미선택: "#555555"
  체크박스_선택: "#f2a900"
  체크박스_미선택: "#333340"
  구분선: "#2a2a35"
  적용_버튼_배경: "#f2a900"
  적용_버튼_텍스트: "#1a1a22"
  초기화_텍스트: "#8b8b8b"
  padding: "좌우 20px, 상하 16px"
  fontSize:
    타이틀: 16
    라벨: 12
    값: 13
```

## 규칙 (위키 사양서)

### 구성
- 타이틀 + 필터 그룹 + 적용/초기화 버튼
- 필터 그룹: 라벨 + 옵션 목록 (체크박스 또는 칩)
- 다중 선택 시 체크박스, 단일 선택 시 라디오

## 적용 케이스 (위키 사양서)

| 화면 | 설명 |
|------|------|
| 커스터마이즈 | 아이템 필터 (등급/유형/컬렉션) |
| 상점 | 상품 필터 |
| 전적 | 통계 필터 |
