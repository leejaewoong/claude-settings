# HoverButton (호버 버튼)

## 용도
호버 시에만 나타나는 플로팅 액션 버튼. 카드 오버레이, 이미지 위 액션.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| 기기 | PC | PC 환경 |
| | Console | 콘솔 환경 |
| 상태 | Nomal | 기본 |
| | Hover | 호버 |
| 사이즈 | Small | 소형 (높이 26px) |
| | Default | 기본 (높이 30px) |
| | Large | 대형 (높이 34px) |

## 규칙 (위키 사양서)

### 플랫폼
- 콘솔: 키 가이드는 버튼 좌측에 배치

### 사이즈
- 가로폭은 최소 너비 이상으로 지정
- 텍스트 길이에 맞춰 가변적으로 적용
- 반드시 텍스트 포함, 줄넘김 없음, 텍스트 길이는 가능한 짧게

### 아이콘
- 텍스트와 아이콘 병행 표시 가능
- 콘솔 조작 키를 제외한 정보 아이콘은 반드시 버튼 우측에 배치
- 아이콘은 자유롭게 수정 가능

## 추천 사이즈

| 사이즈 | PC 너비 | 콘솔 너비 | 높이 |
|--------|---------|-----------|------|
