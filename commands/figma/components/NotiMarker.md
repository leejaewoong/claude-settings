# NotiMarker (노티 마커)

## 용도
아웃게임 내 피쳐의 변화를 알려주는 시그널링 UI. 신규/변경 알림, 보상 수령 알림 표시.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| Type | Normal | 일반 알림 (신규/변경 사항 위치 가이드) |
| | Reward | 보상 알림 (수령 가능한 보상 표시) |
| State | Default | 기본형 (점 형태) |
| | Highlight | 강조형 (디자인적 강조 필요 시) |
| | Count | 수량 표시형 (알림 여러 개일 때) |

## textOverrides

| 키 | 설명 |
|----|------|
| Count | 수량 표시형에서 알림 수 (최대 99, 초과 시 "99+") |

## 규칙 (위키 사양서)

### 타입
- 일반 알림: 신규 및 변경 사항에 대한 위치 가이드가 필요할 때 표시
- 보상 알림: 수령 버튼을 클릭해 획득할 수 있는 보상이 있을 때 표시

### 상태
- 기본형: 기본 점 형태
- 강조형: 디자인적 강조가 필요한 경우
- 수량 표시형: 알림이 여러 개일 경우 상위 UI에 표시
  - 최대 두 자릿수까지 숫자 표기
  - 두 자릿수를 넘을 경우 `+` 기호 함께 표기

## 적용 케이스 (위키 사양서)

| 화면 | 설명 |
|------|------|
| GNB / LNB | 일반, 보상 알림 모두 기본형으로 통일 |
| 글로벌 아이콘 | 일반, 보상 알림 모두 강조형으로 통일 |
| 푸터 키 | 일반 알림 - 기본형 |
| 소셜 위젯 | 일반 알림 - 수량형 |
| 커스터마이즈 | 상단 탭: 일반 알림 강조형 / 사이드 탭: 일반 알림 수량형 |
| 패스 | 상단 탭: 일반+보상 강조형 통일 / 사이드 탭: 일반 기본형, 보상 수량형 |
| 제작소 | 일반 - 기본형 |
| 은신처 | 일반 - 기본형, 사이드 탭: 일반 알림 수량형 |
| OES | 사이드 탭: 일반, 보상 알림 모두 강조형으로 통일 |
