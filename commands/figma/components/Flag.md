# Flag (플래그)

## 용도
상태 플래그, 강조 마커. 한정, 이벤트, 추천/인기 표시.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| Type | Yellow | 골드/한정/추천 |
| | Red | 긴급/인기/HOT |
| | Green | 완료/성공 |
| | Blue | 정보/이벤트 |
| | New | 신규 표시 |

## 규칙 (위키 사양서)

### 텍스트
- 짧게 콘텐츠의 성격을 표기, 문장 사용 금지
- L10N 최소 길이: 184px

### 배치
- 관련 콘텐츠 영역 내부 또는 외부에 표시
- 가급적 영역 상단 좌측으로 배치

### 색상 활용
- 콘텐츠 카테고리가 다른 경우 구별 가능한 다른 색상 활용

## 추천 사이즈

너비 40~184, 높이 24. 카드/아이템 좌상단에 오버레이 배치.

## 적용 케이스 (위키 사양서)

| 용도 | 화면 | 설명 |
|------|------|------|
| 신규 콘텐츠 강조 | 시즈널 맵 로비 배너, 맵 선택 모달 피쳐드 배너 | Type=New |
| 신규 상품 프로모션 | 뉴스 > 프로모션 배너 | |
| 콘텐츠 카테고리 구분 | 뉴스 > 커뮤니티 배너 | 다른 컬러로 구분 |
