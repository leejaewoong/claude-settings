# Tooltip (툴팁)

## 용도
인터페이스 호버 시 콘텐츠 관련 부가 설명을 노출하는 UI. 일반 툴팁과 아이템 툴팁으로 구분.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| Type | General | 일반 툴팁 |
| | Item | 아이템 툴팁 |
| Width | Fixed | 고정 너비 (300px) |
| | Variable | 가변 너비 (원하는 길이 설정) |
| Title | On | 타이틀 표시 |
| | Off | 타이틀 숨김 |
| Divider | On | 분리선 표시 |
| | Off | 분리선 숨김 |
| Scroll | On | 스크롤 표시 |
| | Off | 스크롤 숨김 |

## textOverrides

| 키 | 설명 |
|----|------|
| Title | 툴팁 제목 |
| Description | 툴팁 본문 |

## 규칙 (위키 사양서)

### 일반 규칙
- 특정 인터페이스를 Hover 시 노출 기본, 필요에 따라 Click 노출 가능 (Click은 지양)
- 아이템 슬롯 Hover 시 아이템 툴팁, 그 외는 일반 툴팁 사용
- 내용이 길어질 경우 스크롤 표시

### 일반 툴팁
- 고정 너비: 가로 300px 고정, 콘텐츠량에 따라 세로 확장
- 가변 너비: 원하는 가로 길이 설정, 콘텐츠량에 따라 세로 확장
- 타이틀과 분리선은 독립적으로 On/Off 가능 (동시 사용 가능)
- 높이 고정 시 또는 화면 안전 마진 초과 시 스크롤 표시

### 아이템 툴팁 포함 정보
| 항목 | 기본값 | 설명 |
|------|--------|------|
| 아이템명 | 항상 On | 아이템명 항상 노출 |
| 티어 및 유형 | Off | 데이터 기반 노출, 텍스트 컬러는 티어별 색상 |
| 사용 불가 | Off | 크로마나, 편집 옵션 등 사용 불가 아이템 |
| 성장형 아이템 | Off | 성장형 무기/프레스티지 스킨, 최대 시 Lv.MAX |
| 커스텀 가능 | Off | 커스텀 스페셜 차량 등 |
| 배틀 스탯 | Off | 킬 카운트 (Total Kills/Headshots) |
| 잠금/획득 필요 | Off | 분해 잠금 또는 미보유 옵션 |
| 거래 가능 | Off | 스팀 장터 거래 가능 여부 |
| 출시년도 | Off | 출시년도 표시 |
| 콜렉션 | Off | 컬렉션 이름 (미포함 시 "기타") |
| 부가 정보 | Off | 최대 3개 표시 가능 |
| 기본 설명 | Off | 아이템 설명 |
| 타임 인디케이터 | Off | 기간제 아이템 남은 시간 |

### 아이템 툴팁 스크롤
- 기본적으로 스크롤 미표시
- 화면 안전 마진 영역 초과 시에만 스크롤 표시
