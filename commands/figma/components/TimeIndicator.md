# TimeIndicator (타임 인디케이터)

## 용도
아이템 유효 기간, 이벤트 기간 등 남은 시간을 표시하는 컴포넌트.

## Variant 속성

| 속성 | 값 | 설명 |
|------|-----|------|
| Type | Default | 기본형 ({0}d {1}h {2}m) |
| | Compact | 축약형 (하나의 시간 단위만 표기) |
| | Icon | 아이콘형 (협소 영역용) |
| Size | Small | 소형 |
| | Medium | 중형 |
| | Large | 대형 |
| | XLarge | 초대형 |
| State | Normal | 일반 (남은 시간 7일 이상) |
| | Highlight | 하이라이트 (남은 시간 7일 이하) |
| | Expired | 만료 (남은 시간 없음) |

## textOverrides

| 키 | 설명 |
|----|------|
| Time | 시간 표시 텍스트 |
| Prefix | 좌측 추가 문구 (예: "ENDS IN") |
| Suffix | 우측 추가 문구 |

## 규칙 (위키 사양서)

### 시간 갱신
- 1초마다 동기화되어 자동 갱신

### 시간 표기
- d/h/m은 소문자 표시 (숫자와 혼동 방지)
- 상위 시간이 0이면 상위 시간 생략 (0d → 59h 59m 표기)
- 1분 이하일 때 s(초단위) 표기

### 기본형 시간 포맷
- `{0}d {1}h {2}m` / `{0}h {1}m` / `{0}m` / `{0}s`
- 남은 시간에 따라 자동 선택

### 축약형 시간 포맷
- 하나의 시간 단위로만 간략 표기
- 남은 시간 ÷ 24h → `{0}d`
- 남은 시간 < 24h → `{0}h`
- 남은 시간 < 1h → `{0}m`
- 남은 시간 < 1m → `{0}s`

### 상태 자동 전환
- 일반 → 하이라이트 → 만료 의 3단계 스타일 자동 적용
- 남은 시간 7일 이상: Normal, 7일 이하: Highlight, 0: Expired

### 기타
- 시간 외 매개 변수 `{0}` 사용하지 않음
- 기본형 앞/뒤에 간단한 문구 추가 가능
- 태그 스타일(배경) 적용 가능 (텍스트식 vs 태그식)
- 범용 문구: "ENDS IN", "DDD"

## 적용 케이스 (위키 사양서)

| 화면 | 피쳐 | 타입 |
|------|------|------|
| 밀수품 리스트 | 은신처 | 축약형 태그식 |
| 밀수품 상세 정보 | 은신처 | 기본형 텍스트식 |
| 부가 재화 유효 기간 | - | 축약형 텍스트식 |
| 아이템 슬롯 (기간제) | 커스터마이즈/미리보기/제작소 등 | 아이콘형 |
| 아이템 툴팁 (기간제) | 커스터마이즈/미리보기/제작소 등 | 기본형 텍스트식 |
| 남은 상품 판매 기간 | 상점 미리보기 | 기본형 텍스트식 |
| 패스 배너 남은 기간 | 패스 | 기본형 텍스트식 |
