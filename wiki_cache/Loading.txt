# [사양서] Loading UI

## 정의
 
화면을 불러올 준비로 사용하는 UI
[image]
## UX Guide
- 콘텐츠 로딩이 진행 중임을 명확히 전달 할때 사용하는 피드백 UI

- 서버와 연결 중으로 데이터를 불러오는 등 사용자에게 기다림을 요구할 때 사용
