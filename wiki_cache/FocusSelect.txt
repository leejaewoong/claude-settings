# [사양서] Focus / Select

## 정의 
 아웃게임에서의 포커싱/호버 상태와 셀렉트를 확인 할 수 있는 2px의 라인 박스
[image]
## UX Guide

- Focus (Console) : 포커스 상태를 표시

- Hover (PC) : 마우스 호버 시 호버 액션으로 사용 가능함

- Select (PC/Console) : 선택 상태의 UI를 보여줌

## Spec
- 동일 선상에서는 Focus 규칙에 따라 Select 보다 상위에 Focus 라인이 위치합니다.

## Property
- ***이미지**를 최대한 잘 활용해서 넣자*

- *의도 설명 불필요, 스펙만 나열*

- *아무것도 정의하지 않았을 때 기본값으로 노출되는 상태에 * *표시*
